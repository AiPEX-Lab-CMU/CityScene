8K Skybox Pack Free
-----------------------

To test the skyboxes please simply open the sample scenes in the folder "Example Scenes".

Please note:
* On Android platform, the maximum allowed cubemap size is 2048.
* On iOS platform, the maximum allowed cubemap size is 4096.
* In this package, the "max size" attribute of all cubmap textures is set to 2048 by default.  If you want to use the skyboxes on PC platform or iOS platform, please change the "max size" attribute of the cubemap textures to 4096.
